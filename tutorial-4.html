Van blok-level naar inline (v.v.)

Je kunt een block-level element weergeven als een inline-element (of omgekeerd):

  • display: inline; - geef element inline weer.
  • display: block; - geef element als block-level element weer.
    Dit is de default-waarde voor block-level elementen zoals een <p>-element.
  • display: none; - geef het element niet weer.

Deze style-regels veranderen niets aan de html-regels: deze laatste bepalen welke elementen je binnen een ander element kunt plaatsen. Een <p>-element blijft voor html een block-level element.
