Voor het positioneren kunnen we de volgende middelen gebruiken:

  • width, height - grootte van de box;
  • display - manier van weergeven
    block, inline, hidden
  • float - "drijvend" naar een bepaalde kant
    left, right, top, bottom
  • clear - "geen elementen ernaast"
    left, right, both

De combinatie van deze aspecten en de eigenschappen van het venster bepaalt het gedrag. Soms moet je meerdere van deze aspecten tegelijk veranderen om het gewenste gedrag te krijgen.

Van blok-level naar inline (v.v.)

Je kunt een block-level element weergeven als een inline-element (of omgekeerd):

  • display: inline; - geef element inline weer.
  • display: block; - geef element als block-level element weer.
    Dit is de default-waarde voor block-level elementen zoals een <p>-element.
  • display: none; - geef het element niet weer.

Deze style-regels veranderen niets aan de html-regels: deze laatste bepalen welke elementen je binnen een ander element kunt plaatsen. Een <p>-element blijft voor html een block-level element.

Groottes

Je kunt de groottes in CSS op verschillende manieren aangeven:

  • in pixels (20px) - vooral display-gerelateerd;
  • in em (12em) - gerelateerd aan huidige lettergrootte;
    Dit is de grootte van de letter "m" in het huidige lettertype.
  • in procenten (80%) - gerelateerd aan omvattende box.
