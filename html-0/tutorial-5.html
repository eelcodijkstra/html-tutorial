Scheiden van html-elementen en opmaak

Het is niet handig om de opmaak (css-declaraties) bij de elementen zelf te plaatsen. Je kunt deze beter apart houden, in het head-gedeelte, of, nog beter, in een apart bestand.

We zullen hier zien hoe je de css-declaraties kunt scheiden van de elementen. We plaatsen deze declaraties hier in het head-gedeelte. In een latere module zullen we deze in aparte bestanden onderbrengen.

Om de css-declaraties te kunnen koppelen aan de elementen waar deze bij horen, gebruiken we css-selectoren. We zullen hier een paar voorbeelden zien. In een latere module behandelen we deze uitgebreider.

CSS-selectoren: tag

Soms wil je alle elementen van een bepaald type eenzelfde opmaak-eigenschap geven. Bijvoorbeeld: alle kopjes (h1) van eenzelfde lettertype en eenzelfde kleur.

Dit kun je doen door de tag-name als css-selector te gebruiken. Voorbeeld: 

        h1 {
          font-family: sans-serif;
          color: lightblue;
        }

De notatie hier is: css-selector { css-declaraties }

Merk op hoe in het eerste deel van het voorbeeld de html-code en de opmaak gescheiden zijn. Dit verbetert de leesbaarheid, en je kunt de vormgeving eenvoudig aanpassen: alles staat bij elkaar.

Opdrachten

  1. Voeg aan het style-gedeelte in het head-deel css-declaraties toe om (i) alle p-elementen in een bepaald soort lettertype te zetten; (ii) alle em-elementen te onderstrepen.
  2. Als je voor meerdere soorten elementen eenzelfde stijl wilt gebruiken, dan kun je de verschillende tags opsommen gescheiden door een comma. Bijvoorbeeld: h1, h2 {...}.
    Pas het voorbeeld aan zodat de stijl van h1 ook voor h2 en h3 geldt.

Groepering: het div-element en css id-selector

We hebben eerder gezien hoe je elementen in lopende tekst kunt groeperen met het span-element. Met het div-element kun je andere elementen groeperen.

Dit groeperen is vooral handig als je de stijl (opmaak) van een groep elementen wilt aangeven: je groepeert dan de elementen in een div-element, en geeft de stijl aan voor dit div-element.

Je kunt een stijl aan een bepaald element koppelen door dit element een naam te geven. Voorbeeld: <div id="mydivname"> ... . Een dergelijke naam moet uniek zijn: je mag eenzelfde naam niet voor meerdere elementen gebruiken.

De bijbehorende css-selector voor het id-attribuut is in dit voorbeeld: #mydiv.

Opdrachten

  1. Bekijk in het voorbeeld hoe de css-declaraties van een div-element in het head-deel geplaatst kunnen worden, door dit element een unieke naam (id-attribuut) te geven. Doe dit ook voor de andere elementen, zodat alle css-declaraties in het head-deel komen te staan.
  2. Als je voor meerdere soorten elementen eenzelfde stijl wilt gebruiken, dan kun je de verschillende tags opsommen gescheiden door een comma. Bijvoorbeeld: h1, h2 {...}.
    Pas het voorbeeld aan zodat de stijl van h1 ook voor h2 en h3 geldt.

De box die een element omvat, bestaat (van buiten naar binnen) uit:

  • margin (marge)
  • border (rand)

Van blok-level naar inline (v.v.)

Je kunt een block-level element weergeven als een inline-element (of omgekeerd):

  • display: inline; - geef element inline weer.
  • display: block; - geef element als block-level element weer.
    Dit is de default-waarde voor block-level elementen zoals een <p>-element.
  • display: none; - geef het element niet weer.

Deze style-regels veranderen niets aan de html-regels: deze laatste bepalen welke elementen je binnen een ander element kunt plaatsen. Een <p>-element blijft voor html een block-level element.

  • Voeg binnen de style-waarde van de binnenste <p>-elementen toe: display: inline;.
