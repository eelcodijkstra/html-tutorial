Scheiden van html-elementen en opmaak

Het is niet handig om de opmaak (css-declaraties) bij de elementen zelf te plaatsen. Je kunt deze beter apart houden, in het head-gedeelte, of, nog beter, in een apart bestand.

We zullen hier zien hoe je de css-declaraties kunt scheiden van de elementen. We plaatsen deze declaraties hier in het head-gedeelte. In een latere module zullen we deze in aparte bestanden onderbrengen.

Om de css-declaraties te kunnen koppelen aan de elementen waar deze bij horen, gebruiken we css-selectoren. We zullen hier een paar voorbeelden zien. In een latere module behandelen we deze uitgebreider.

CSS-selectoren: type

Soms wil je alle elementen van een bepaald type eenzelfde opmaak-eigenschap geven. Bijvoorbeeld: alle kopjes (h1) van eenzelfde lettertype en eenzelfde kleur.

Dit kun je doen door de tag-name als css-selector te gebruiken. Voorbeeld: 

        h1 {
          font-family: sans-serif;
          color: lightblue;
        }

De notatie is: css-selector { css-declaraties }

In het eerste deel van het voorbeeld zijn de html-code en de opmaak gescheiden. Dit verbetert de leesbaarheid, en je kunt de vormgeving eenvoudig aanpassen: alles staat bij elkaar.

Opdrachten

  1. Voeg aan het style-gedeelte in het head-deel css-declaraties toe om (i) alle p-elementen in een bepaald soort lettertype te zetten; (ii) alle em-elementen te onderstrepen.
  2. Als je voor meerdere soorten elementen eenzelfde stijl wilt gebruiken, dan kun je de verschillende tags opsommen gescheiden door een comma. Bijvoorbeeld: h1, h2 {...}.
    Pas het voorbeeld aan zodat de stijl van h1 ook voor h2 en h3 geldt.

Groepering: het div-element en css id-selector

We hebben eerder gezien hoe je elementen in lopende tekst kunt groeperen met het span-element. Met het div-element kun je andere elementen groeperen.

Dit groeperen is vooral handig als je de stijl (opmaak) van een groep elementen wilt aangeven: je groepeert dan de elementen in een div-element, en geeft de stijl aan voor dit div-element.

Je kunt een stijl aan een bepaald element koppelen door dit element een naam te geven. Voorbeeld: <div id="mydiv"> ... . Een dergelijke naam moet uniek zijn: je mag een naam maar voor één element gebruiken.

De css-selector voor het zo benoemde element is: #mydiv.

Opdrachten

  1. Bekijk hoe je de css-declaraties van een div-element in het head-deel kunt plaatsen, door dit element een unieke naam (id-attribuut) te geven (vb: mydiv1). Doe dit ook voor de andere elementen, zodat alle css-declaraties in het head-deel komen te staan.
  2. Als je voor meerdere soorten elementen eenzelfde stijl wilt gebruiken, dan kun je de verschillende tags opsommen gescheiden door een komma. Bijvoorbeeld: h1, h2 {...}.
    Pas het voorbeeld aan zodat de stijl van h1 ook voor h2 en h3 geldt.

Pagina-opmaak: het box-model

Voor de opmaak rond een element gebruikt CSS het box-model. De box die een element omvat, bestaat van buiten naar binnen uit:

  • margin (marge - buiten de box)
  • border (rand - grens tussen binnen en buiten de box)
  • padding (opvulling - binnen de box)
  • de eigenlijke inhoud

Van de margin, border en padding kun je de grootte aanpassen, eventueel per kant. De kleur of achtergrond van de box of de rand kun je ook bepalen. Hiermee kun je de directe omgeving van de inhoud van een element vormgeven.

Opdrachten

  1. Verander van het div-element mydiv1 de verschillende waarden voor margin, border, en padding, en bekijk het resultaat.
  2. Je kunt de afstand tussen de blauwe div-box en de rode div-box daarbinnen aanpassen door (i) de padding van de blauwe box te vergroten; (ii) de margin van de rode box te vergroten. Maakt dit verschil?
  3. Experimenteer met de padding waarden voor de rode box, om iets te kunnen zeggen over de margin van de p-boxen. Wat is je conclusie?

Kleur en achtergrond

We kunnen nog meer eigenschappen van een box aanpassen:

  • border - hiervan kun je de vorm en de kleur aangeven, bijvoorbeeld solid blue of "gestippeld rood".
  • background-color - de achtergrondkleur binnen de box. De margin en de border vallen hierbuiten. De achtergrond van de margin wordt bepaald door de achtergrond van de omvattende box.

Opdracht(en)

  • Geef het element mydiv1 de achtergrondkleur lightblue. Wat kun je zeggen over de achtergrondkleur van de elementen waarvan je de kleur niet opgegeven hebt?
  • Geef het div-element met de rode rand de achtergrondkleur lightpink, en het eerste p-element daarin de achtergrondkleur lightyellow. Wat kun je zeggen over de achtergrondkleur van de margin van een element?

Van blok-level naar inline (v.v.)

Je kunt een block-level element weergeven als een inline-element (of omgekeerd):

  • display: inline; - geef element inline weer.
  • display: block; - geef element als block-level element weer.
    Dit is de default-waarde voor block-level elementen zoals een <p>-element.
  • display: none; - geef het element niet weer.

Deze opmaak-regels veranderen niets aan de html-regels: deze laatste bepalen welke elementen je binnen een ander element kunt plaatsen. Een <p>-element blijft voor html een block-level element.

  • Voeg aan de css-declaraties van de <p>-elementen binnen de div-elementen toe: display: inline;. Wat is hiervan het effect?
  • Waarom zou je een element niet weer willen geven?
    (Hint: je kunt een dergelijke eigenschap besturen door middel van Javascript in de betreffende pagina.)
